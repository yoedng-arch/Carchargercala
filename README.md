# ⚡ עלות טעינה EV

Simple, fast charging cost calculator for Geely Geometry C 2024.

## Features
- 📱 **Mobile-first**: 100% optimized for mobile devices
- 🎨 **Modern design**: Dark theme with sharp cyan/teal gradients
- 💰 **Two-stage input**: Start charge → End charge
- 📊 **Auto calculation**: Real-time cost, energy, range metrics
- 📤 **WhatsApp sharing**: Share charging details with one tap
- ⌨️ **Keyboard support**: Press Enter to move between stages

## Specifications
- **Vehicle**: Geely Geometry C 2024
- **Battery**: 70 kWh
- **Rate**: ₪0.65/kWh
- **No extras**: Clean, purpose-built interface

## How to Use
1. Enter starting battery % and km remaining
2. Tap "Continue"
3. Enter ending battery % and km remaining
4. Tap "Calculate"
5. View cost breakdown or share on WhatsApp

## Tech Stack
- Pure HTML5
- CSS3 (gradients, animations, RTL support)
- Vanilla JavaScript (no dependencies)
- Mobile responsive (viewport optimized)

## Deploy
**Netlify (recommended):**
- Drag `index.html` to [netlify.com](https://netlify.com)
- Get live URL in seconds

**Or use Netlify CLI:**
```bash
npm install -g netlify-cli
netlify deploy
```
